# Personal Workspace Template

Simple content creation workspace created during first-run setup.

## When to Use

Use this template when running `/cofounder/system/Start Here.md` setup. Creates a minimal personal workspace for content creation.

## What's Included

- `AGENTS.md` - Workspace instructions (references Content Author and voice settings)

## vs Workspace Root Template

| Personal Workspace | Workspace Root |
|-------------------|----------------|
| Simple `/content/` directory | Complex structure (clients, marketing, internal) |
| Individual content creator | Team or business workspace |
| Created during setup | Created for new projects |

## Usage

1. Copy `AGENTS.md` to `/[your name]/AGENTS.md`
2. Replace `[Your Name]` with your actual name
3. Delete the template instructions section
