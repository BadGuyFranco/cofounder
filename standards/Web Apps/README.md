# Web Apps

Coding standards and architecture patterns for building full-stack web applications.

## Documentation

- **AGENTS.md** - Complete instructions for AI agents (discovery, scaffolding, templates)

## Key Features

- **Requirements Discovery** - Guided process to understand what you're building
- **Full Project Scaffolding** - Generate complete monorepo structure
- **AI Assistant Built-in** - Every project includes AI overlay architecture
- **Multi-Tenancy Ready** - Optional tenant isolation pattern
- **Documentation First** - Context files that make AI collaboration effective

## Tech Stack

| Technology | What It Does |
|------------|--------------|
| **React** | UI component library for building interactive interfaces |
| **TypeScript** | Typed JavaScript that catches errors before runtime |
| **Zustand** | Lightweight state management without boilerplate |
| **Tailwind CSS** | Utility classes for styling directly in markup |
| **Express** | Minimal Node.js framework for building APIs |
| **PostgreSQL** | Relational database with JSON support |
| **Prisma** | Type-safe database client with automatic migrations |
| **Clerk** | Managed authentication service handling login, signup, and sessions |
| **Zod** | Schema validation for runtime type checking |

## Common Use Cases

Use these standards when starting a new web application or SaaS. The process guides you through requirements gathering, architecture decisions, and generates a complete project with documentation that enables effective AI collaboration.
