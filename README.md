# Shared Tools

A collection of AI-powered tools for content creation, image/video generation, and productivity.

## Getting Started

1. Clone this repository
2. Open in Cursor
3. Follow `system/Start Here.md` to set up your workspace

## What's Included

- **Content Author** - Write in your authentic voice
- **Image Generator** - Generate images with AI
- **Video Generator** - Create videos from images and prompts
- **Visualizer** - Create diagrams and visual explanations
- **Marketing System** - Marketing frameworks and copy
- **And more...**

See `AGENTS.md` for the full tool list and routing guide.

## Structure

```
/pro accelerator/
├── tools/         # AI-powered libraries
├── system/        # Setup and maintenance
├── memory/        # Your personal config (gitignored)
└── [your name]/   # Your content workspace (created during setup)
```
