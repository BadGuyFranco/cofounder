# CourtListener Connector

Connect to CourtListener (Free Law Project) via their REST API for legal citation verification and case-law research.

## Status

Active - citation lookup, opinion search, and full-text retrieval are working and live-tested.

## Features

- Verify that a citation resolves to a real case
- Search opinions and case law, with court filters
- Retrieve case metadata and full opinion text
