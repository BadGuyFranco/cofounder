# Figma Connector

Connect to Figma via their REST API.

## Status

Active - Full read access to files, images, comments, components, and styles.

## Features

- Read file content and structure as JSON
- Export frames and components as PNG, JPG, SVG, or PDF
- Download exports directly to local files
- Manage comments and reactions on files
- Access published components and styles from libraries
- Verify token setup with user info
