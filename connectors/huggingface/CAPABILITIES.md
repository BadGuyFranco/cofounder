# HuggingFace Connector Capabilities

What this connector can do for you.

## Models

- Browse and search the model hub
- Get model details, files, and metadata
- Download models to local storage
- Download specific model files
- Upload models to your account
- List your models

## Datasets

- Browse and search datasets
- Get dataset details and metadata
- Download datasets to local storage
- Download specific dataset files
- List your datasets

## Repositories

- Create new model, dataset, or space repositories
- Delete repositories you own
- List your repositories
- Update repository settings (visibility, tags)

## Inference (Run Models)

- Generate text from prompts
- Generate images from text
- Generate embeddings for semantic search
- Classify text (sentiment, topic)
- Summarize documents
- Transcribe audio to text
- Run any model on the Inference API

## Account

- Verify credentials
- View account information
- List organizations you belong to

## Limitations

- Large model downloads can take significant time and disk space
- Some models/datasets are gated (require accepting terms first)
- Free tier has rate limits on inference
- Cold start delays on less popular models (20-60 seconds)
- Write operations require write-enabled token
- Some models require HuggingFace Pro for reliable inference access
