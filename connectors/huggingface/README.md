# HuggingFace Connector

Connect to HuggingFace to run machine learning models via their Inference API.

## Status

**Active** - Run inference, browse models, generate embeddings.

## Features

- Run any model on HuggingFace Hub
- Text generation, image generation, embeddings
- Browse and search models by task
- Automatic retry for cold-starting models
