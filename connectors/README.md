# Connectors

Connect to external platforms via their APIs.

## Documentation

- **AGENTS.md** - Complete instructions for AI agents (routing, setup, usage)

## Key Features

- **Platform integrations** - Notion, HubSpot, Monday.com, Go High Level, Zapier, Make, Airtable
- **Unified patterns** - Consistent API handling, error patterns, and configuration across platforms
- **Optional setup** - Only configure the connectors you need

## Common Use Cases

Push content to Notion pages or databases. Sync data with CRM platforms like HubSpot or Go High Level. Query project management tools like Monday.com or Airtable.
