# Monday.com Connector Capabilities

What this connector can do for you.

## Boards

- Create new boards
- Duplicate existing boards
- Archive and delete boards
- View board structure

## Items

- Create items (rows)
- Update item values
- Move items between groups
- Delete items

## Groups

- Create groups within boards
- Rename and reorder groups
- Delete groups

## Columns

- Add new columns (fields)
- Update column settings
- Delete columns

## Subitems

- Create subitems under items
- Update and delete subitems

## Comments

- Add comments to items
- View activity feed
- Delete comments

## Workspaces

- Create workspaces
- Manage workspace settings

## Search

- Search items across boards
- Search boards by name

## Users

- View team members
- Get user information

## Tags

- Create and manage tags
- Apply tags to items

## Limitations

- Complexity-based rate limits (10M points/minute)
- Column value updates require specific JSON format
- Subitems have separate board structure
