# Make Connector

Connect to Make.com to manage scenarios, webhooks, and data stores.

## Status

**Active** - Scenario and webhook management with data store operations.

## Features

- List and run scenarios
- Trigger webhooks
- Manage data stores
- Export/import blueprints
- View execution history

https://developers.make.com/api-documentation

## Note: Free Plan Limitations

Many API features require a paid Make.com plan. Free accounts can list resources and trigger webhooks but cannot access scenario details or data store records via API.
