# Airtable Connector Capabilities

What this connector can do for you.

## Records

- Create records in any table
- Read and search records
- Update existing records
- Delete records
- Filter records by any field

## Tables

- View table structure and fields
- List all tables in a base

## Bases

- List all your bases
- View base schema

## Fields

- Create new fields (columns)
- Update field settings

## Comments

- Add comments to records
- Update and delete comments

## Webhooks

- Create webhooks for record changes
- Manage webhook subscriptions

## Limitations

- 5 requests per second per base
- Field names are case-sensitive
- Formulas and rollups are read-only
- Attachments require URL (no direct upload)
