# Airtable Connector

Connect to Airtable bases and tables via the Web API.

## Status

**Active** - Full CRUD operations on records, base/table schema reading.

## Features

- Manage records (create, read, update, delete)
- Query and filter data
- View base and table schemas
- Manage webhooks
