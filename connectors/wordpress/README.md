# WordPress Connector

Connect to WordPress sites via the REST API.

## Status

Active - Full CRUD support for posts, pages, media, comments, categories, tags, and users.

## Features

- Multi-site support (manage multiple WordPress sites)
- Posts and pages with scheduling
- Media uploads (images, videos, documents)
- Comment moderation
- Category and tag management
- User information

## Requirements

- WordPress 5.6 or later
- HTTPS enabled
- Permalinks enabled (not "Plain")
