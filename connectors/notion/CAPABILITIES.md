# Notion Connector Capabilities

What this connector can do for you.

## Pages

- Create new pages
- Update page content
- Archive pages
- Search pages

## Databases

- Query database entries
- Create new entries
- Update entry properties
- Filter and sort results

## Content

- Add text, headings, lists
- Add to-do items
- Add code blocks
- Add callouts and quotes

## Limitations

- Integration must be manually connected to each page/database in Notion
- Property names are case-sensitive
- Some block types are read-only
