# Zoom Connector

Connect to Zoom via their REST API.

## Status

Active - Full coverage of Zoom Meeting API v2

## Features

- User management (list, create, update, delete)
- Meeting management (schedule, update, delete, end)
- Webinar management (schedule, registrants, panelists, polls)
- Cloud recording access and management
- Reports and analytics
- Dashboard and real-time metrics
- Group management
