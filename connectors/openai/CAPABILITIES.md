# OpenAI Connector Capabilities

What this connector can do for you.

## Chat & Generation

- Send messages to GPT-4o, GPT-4o-mini, o3-mini, o1 and get responses
- Use a system prompt to set role, tone, or constraints
- Request JSON-formatted output
- Control temperature and output length
- List all available models (including fine-tuned)

## Limitations

- No streaming support (response returned in full)
- No image/vision input in this connector
- No embeddings, DALL-E image generation, or Whisper audio in this connector (planned)
- o1 and o3 models do not support temperature or system messages
- Rate limits vary by usage tier
- No function/tool calling in this connector
