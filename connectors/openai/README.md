# OpenAI Connector

Connect to OpenAI's GPT and o-series models via their API.

## Status

Active — chat completions, model listing.

## Features

- Chat with GPT-4o, GPT-4o-mini, o3-mini, and o1
- System prompt support
- JSON output mode
- Token usage reporting
