# Google Connector

Comprehensive connector for Google services: Drive, Workspace (Docs, Sheets, Slides), Gmail, YouTube, Calendar, and AI (Gemini).

## Status

Active - Full implementation of all modules.

## Features

- Full Google Drive access (files, folders, sharing, comments)
- Google Workspace integration (Docs, Sheets, Slides)
- Gmail send, read, and management
- YouTube video and channel management
- Google Calendar event management
- AI capabilities via Gemini (text, image, video generation)
- Multi-account support
- Automatic account detection from file paths

## Setup

See `SETUP.md` for complete Google Cloud Console configuration.

Quick summary:
1. Create Google Cloud project
2. Enable required APIs
3. Configure OAuth consent screen
4. Create OAuth credentials (Desktop app)
5. Get Gemini API key from AI Studio (for AI features)
6. Run auth setup
