# Cloudflare Connector

Connect to Cloudflare via their API.

## Status

Active - Full DNS, Page Rules, and Zone management support.

## Features

- Manage zones (domains) in your Cloudflare account
- Create, update, and delete DNS records
- Configure page rules and redirects
- Purge cache
- Manage SSL/TLS settings
- Configure firewall rules
- Deploy and manage Workers
