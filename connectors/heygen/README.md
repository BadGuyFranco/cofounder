# HeyGen Connector

Connect to HeyGen's API to generate AI avatar videos.

## Status

Active - Full support for video generation, avatar management, and templates.

## Features

- Create AI avatar videos from text scripts
- Use stock or custom avatars
- Generate videos from templates
- Multiple account support
- Check video status and download completed videos
