# Anthropic Connector Capabilities

What this connector can do for you.

## Chat & Generation

- Send messages to Claude and get responses
- Set a system prompt for role, tone, or constraints
- Control output length (up to 8192 tokens per response)
- Control temperature (0–1 scale)
- Choose between Claude 3.5 Sonnet, Haiku, and Opus

## Discovery

- List all currently available Claude models

## Limitations

- No streaming support (response returned in full)
- No image/vision input in this connector
- No file upload support
- Temperature range is 0–1 (narrower than OpenAI/xAI)
- Rate limits depend on usage tier (Tier 1–4 based on spend history)
- No function/tool calling support in this connector
