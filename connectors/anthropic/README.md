# Anthropic Connector

Connect to Anthropic's Claude models via the Messages API.

## Status

Active — chat completions, model listing.

## Features

- Chat with Claude 3.5 Sonnet, Haiku, and Opus
- System prompt support
- Token usage reporting
- Supports long context windows (up to 200K tokens input)
