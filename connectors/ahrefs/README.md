# Ahrefs Connector

Connect to Ahrefs via their API v3.

## Status

Active. Domain Rating works for free with no account; deeper data (snapshots, referring domains, anchor text, organic keywords) requires a paid API key. Browser Control remains the fallback for data this connector does not expose.

## Features

- Domain Rating lookups for any domain, with competitor benchmarking, free and keyless via Ahrefs' public endpoint
- Full domain snapshot: DR, backlink stats, organic traffic, keywords, and traffic value (paid key)
- Top referring domains and anchor text distribution (paid key)
- Organic keyword rankings with quick-win position filtering (paid key)
- Connectivity check that confirms the free endpoint and validates a paid key if present
