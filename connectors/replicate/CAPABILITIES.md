# Replicate Connector Capabilities

What this connector can do for you.

## Run Models

- Generate images from text prompts
- Generate videos from text or images
- Remove backgrounds from photos
- Upscale and enhance images
- Transcribe audio to text
- Run any public model on Replicate

## Manage Predictions

- Create new predictions (model runs)
- Check prediction status
- Download prediction outputs
- Cancel running predictions
- List recent predictions

## Browse Models

- Get model details and documentation
- View input parameters and defaults
- List model versions
- Browse featured collections

## Download Outputs

- Automatically download generated files
- Save to specified directory
- Support for images, videos, and other outputs

## Limitations

- No model training (inference only)
- Some models require payment method on file
- Video generation can take several minutes
- Rate limits apply during high usage
- Output files are temporary (download promptly)
