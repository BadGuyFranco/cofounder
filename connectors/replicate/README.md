# Replicate Connector

Connect to Replicate to run machine learning models via their API.

## Status

**Active** - Run predictions, browse models, download outputs.

## Features

- Run any public model on Replicate
- Generate images, videos, audio, and more
- Browse model catalogs and versions
- Automatic output downloading
