# Go High Level Connector Capabilities

What this connector can do for you.

## Contacts

- Create, update, and delete contacts
- Search contacts
- Add tags to contacts
- Add notes to contacts

## Opportunities (Deals)

- Create and manage deals
- Move deals through pipeline stages
- Track deal values

## Invoices

- Create invoices
- Send invoices to contacts
- Void and delete invoices

## Products

- Manage product catalog
- Set pricing

## Messaging

- Send SMS messages
- Send emails
- View conversation history

## Appointments

- View calendars
- Book appointments
- Cancel appointments

## Automation

- Trigger workflows
- Add/remove contacts from campaigns

## Tasks

- Create and manage tasks
- Assign tasks to team members

## Other

- Manage custom fields
- Manage tags
- Manage team users
- View form submissions
- Manage social media posts
- Upload media files

## Limitations

- 100 requests per 10 seconds per location
- 200,000 requests per day per location
- Multi-location support (specify which sub-account)
