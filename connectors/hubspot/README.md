# HubSpot Connector

Connect to HubSpot CRM to manage contacts, companies, deals, and their relationships.

## Status

**Active** - Full CRM operations including contacts, companies, deals, engagements, and associations.

## Features

- Manage contacts, companies, deals, tickets
- Log engagements (calls, emails, meetings, notes, tasks)
- Create and manage associations between objects
- List management and workflow enrollment
