# Conversion Expert Capabilities

What this tool can do for you.

## Diagnose

- Find where visitors drop off in your funnel (GA4): pages, steps, sources, devices
- Explain why they drop (Clarity): rage clicks, dead clicks, quick backs, excessive scroll, script errors, shallow scroll past the CTA
- Quantify speed as a conversion cost (Core Web Vitals on your key pages)
- Audit on-page conversion heuristics: value proposition, primary CTA, form friction, trust signals, mobile usability, message-to-source match
- Works even with no connectors configured (heuristic audit of the rendered pages)

## Prioritize

- A single todo list scored by ICE (Impact, Confidence, Ease), sorted by what moves revenue most
- Every item cites the evidence behind it (GA4 drop %, Clarity signal + count, or labeled heuristic)
- Predicts the conversion impact and recommends ship-direct vs A/B test

## Generate

- Rewritten CTA and headline copy grounded in your data
- Shorter-form specs and field-friction fixes
- Trust-signal blocks
- A/B test specifications (hypothesis, variants, success metric)

## Execute and measure

- Set up and track experiments in `cro/experiments/`
- Route build work to the Designer tool and Web Apps standard
- Store dated snapshots so trends survive (Clarity only keeps the last 1 to 3 days)
- Close the loop: each cycle measures the last cycle's changes

## Boundaries

- Acquisition, rankings, and traffic are SEO Expert's job; this tool starts once visitors arrive
- Session recordings and heatmap images are watched in the Clarity UI; the tool flags which pages and sessions are worth opening
- Never fabricates a metric; unavailable data is labeled, not invented
- Live-revenue path edits (checkout, signup) default to staged changes or A/B tests with a stated rollback
