# Video Generator

Generate and edit videos using AI services (Google Veo, Replicate Minimax) and local tools.

## Features

- **AI Video Generation** - Create videos from text prompts
- **Image-to-Video** - Animate still images into video clips
- **Multiple Providers** - Google Veo 3.1 and Replicate Minimax
- **Local Editing** - Trim, concatenate, add text, resize videos
