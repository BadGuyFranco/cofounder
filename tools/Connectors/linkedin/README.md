# LinkedIn Connector

Post content to LinkedIn profiles and company pages via the Marketing API.

## Status

**Active** - Full implementation with authentication, posts, media, organizations, comments, and reactions.

## Features

- Create and manage posts (text, images, videos)
- Post to company pages
- Comment and react to content
- View profile and organization info
