# Go High Level Connector

Connect to Go High Level CRM to manage contacts, opportunities, and business operations.

## Status

**Active** - Comprehensive CRM operations with multi-location support.

## Features

- Manage contacts, companies, opportunities
- Messaging (SMS, email, conversations)
- Calendars and appointments
- Invoices, payments, products
- Social media planner
- Multi-location support
