# Monday.com Connector

Connect to Monday.com project management via GraphQL API.

## Status

**Active** - Full board and item management with subitems, columns, and updates.

## Features

- Manage boards, items, groups, columns
- Subitems (nested items)
- Comments and activity updates
- Workspaces and tags
- Cross-board search
