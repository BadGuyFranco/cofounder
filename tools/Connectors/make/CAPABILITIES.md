# Make Connector Capabilities

What this connector can do for you.

## Scenarios

- List all your scenarios
- Start and stop scenarios
- Run scenarios manually
- View scenario status

## Webhooks

- List webhooks
- Trigger webhooks with data

## Data Stores

- Read data store records
- Add and update records
- Delete records

## Blueprints

- Export scenario blueprints (backup)
- Import blueprints
- Clone scenarios

## Connections

- List app connections
- Verify connection status

## Analytics

- View execution history
- Check usage statistics

## Organization

- List teams and organizations
- Manage folders

## Limitations

- Many features require paid Make plan
- Free plan: Can list and trigger webhooks only
- Paid plan required for: scenario details, blueprints, data stores, analytics
