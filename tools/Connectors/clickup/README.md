# ClickUp Connector

Connect to ClickUp workspaces, spaces, folders, lists, and tasks via the API.

## Status

**Active** - Full CRUD operations on tasks, lists, folders, spaces. Comments and time tracking supported.

## Features

- Manage tasks (create, read, update, delete)
- Organize with spaces, folders, and lists
- Add comments and track time
- Search tasks across workspaces
- Set custom field values
