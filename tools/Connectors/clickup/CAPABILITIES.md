# ClickUp Connector Capabilities

What this connector can do for you.

## Workspaces

- List all workspaces you have access to
- View workspace details

## Spaces

- List spaces in a workspace
- Create new spaces
- View space details

## Folders

- List folders in a space
- Create new folders
- Update folder names
- Delete folders

## Lists

- List all lists in a folder
- List folderless lists in a space
- Create new lists
- Update list settings
- Delete lists

## Tasks

- Create tasks in any list
- Update task details (name, description, status, priority, due date)
- Assign tasks to team members
- Move tasks between lists
- Delete tasks
- Search tasks across workspace
- List subtasks

## Comments

- Add comments to tasks
- View task comments
- Update and delete comments

## Custom Fields

- View custom field definitions
- Set custom field values on tasks

## Time Tracking

- View time tracked on tasks
- Add time entries

## Limitations

- 100 requests per minute rate limit
- Personal tokens have full account access (no granular scopes)
- Some features require paid plans (Goals, Dashboards)
- Subtask hierarchy limited to one level in API
- Bulk operations limited to 100 items per request
