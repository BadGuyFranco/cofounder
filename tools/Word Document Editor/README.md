# Word Document Editor

Create and edit Word documents with full support for track changes, using stable XML parsing methods.

## Features

- **Create Documents** - Create new Word documents from scratch
- **Read Track Changes** - List all tracked changes in a document
- **Apply/Reject Changes** - Accept or reject specific changes with explicit approval
- **Add Tracked Changes** - Insert new edits as tracked changes (defaults to "AIM" author)
- **Read Comments** - List all comments in a document
- **Add Comments** - Add new comments for review (defaults to "AIM" author)
- **Safety First** - Never accepts changes without explicit user confirmation
