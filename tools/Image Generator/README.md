# Image Generator

Generate and edit images using AI services (Google Gemini, Replicate FLUX Pro) and local tools.

## Features

- **AI Image Generation** - Create images from text prompts
- **Multiple Providers** - Google Gemini and Replicate FLUX Pro
- **Local Editing** - Resize, crop, convert formats, adjust colors
- **Background Removal** - Remove backgrounds from photos
