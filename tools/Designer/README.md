# Designer

Create professional visual design artifacts: color palettes, typography systems, UI components, page layouts, wireframes, design tokens, and style guides.

## Documentation

- **AGENTS.md** - Complete instructions for AI agents

## Key Features

- **Design System Creation** - Build complete design systems with color, typography, spacing tokens and DESIGN.md output
- **Component Design** - Generate production-ready UI components in HTML/Tailwind with all interactive states
- **Page Layout** - Compose full page designs with responsive breakpoints and visual hierarchy
- **Color Palettes** - OKLCH-based palettes with WCAG accessibility validation and light/dark mode support
- **Typography Systems** - Modular scale type systems with font selection and fluid sizing
- **Wireframing** - Structural grayscale layouts with annotations for rapid prototyping
- **Style Guides** - Generate visual documentation of any design system as browsable HTML
- **Quality Validation** - Integrates with Design Advisor for automated quality checks against professional standards

## Common Use Cases

Design new UI components, landing pages, or dashboards with professional graphic design quality. Build design token systems for new projects. Generate color palettes and typography scales. Create wireframes for rapid prototyping. Document existing design decisions as style guides.
