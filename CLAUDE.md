## AGENTS.md Convention

Every directory in this project ecosystem uses `AGENTS.md` as its mandatory first-read entry point. When entering any directory, read its `AGENTS.md` before reading code, architecture docs, or making changes. Follow the routing tables to reach subdirectories, and read their `AGENTS.md` in turn. Never skip this chain.

`CLAUDE.md` files exist only to bridge Claude Code into the AGENTS.md convention. All routing, directory instructions, and project knowledge belong in `AGENTS.md` files. Never add project-specific instructions to `CLAUDE.md`.
