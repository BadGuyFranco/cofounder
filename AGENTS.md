# Shared Tools

AI agent routing for the shared tools library.

## First Time Setup

New users: Follow `/pro accelerator/system/Start Here.md`

## Structure

| Directory | Purpose |
|-----------|---------|
| `/tools/` | AI-powered libraries (Content Author, Image Generator, etc.) |
| `/system/` | Setup guides and maintenance documentation |
| `/memory/` | User configuration (voice, API keys) - gitignored |

## Tool Routing

| Need | Tool |
|------|------|
| Write content | `tools/Content Author/` |
| Generate images | `tools/Image Generator/` |
| Generate videos | `tools/Video Generator/` |
| Create diagrams | `tools/Visualizer/` |
| Transcribe audio | `tools/Voice Transcription/` |
| Marketing frameworks | `tools/Marketing System/` |
| Edit Word docs | `tools/Word Document Editor/` |
| Browser automation | `tools/Browser Control/` |
| Complex problem solving | `tools/Problem Solver/` |
| Review/write prompts | `tools/Prompt Author/` |
| Build web apps | `tools/Web Tech Stack/` |

Each tool has its own `AGENTS.md` with detailed instructions.
